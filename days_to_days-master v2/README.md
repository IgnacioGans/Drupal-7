# Days to Day
Module for Drupal 7
Convert a number to human day
* example
- 0.5 -> 1/2 Day
- 1 -> Full Day
- 2 -> 2 Days
- 2 -> 2 D
- 2 -> 2 D / 1 N
- 2 -> 2 Days / 1 Night
